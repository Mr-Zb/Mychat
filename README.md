### Chat 是一款springboot+webscoket实现的即使通讯系统

## 项目简介

Springboot 1.5.13

登陆：采用security+redis实现登陆

缓存：redis

持久层：mybatis

代码生成：mybatis-generator

Oss:阿里云OSS

vue2.X

uni-app

## 环境

- JDK 1.8
- Intellij IDEA
- mysql 5.7
- git 版本管理
- nginx 反向代理
- redis 缓存
